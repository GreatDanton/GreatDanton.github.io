Dobrodošli!

Stran je bila na začetku namenjena samo zame, za učenje front end web-developmenta (testiranje css frameworkov, jquery modulov ipd). Ker trenutno iščem delo na področju web developmenta, sem se odločil da jo odprem za javnost (delodajalce), saj slika pove več kot 1000 besed. Stran sem napisal sam v html5/css3/jquery, uporabil sem le Kube framework grid system. Spletna stran se prav tako odziva na različne resolucije (Responsive web design). Ko je resolucija zaslona manjša kot 651px se glavni meni skrije in pojavi se nov način izbire podstrani.

Komentarji na podstrani Test so izključno samo prikaz javascripta, saj je potrebno za komentarje imeti bazo podatkov, katere še nisem implementiral. Prav tako je tudi sign-in link namenjen kasnejši implementaciji login sistema. Trenutno je prisotnih še nekaj manjših napak. Isti browserji renderirajo strani na različnih operacijskih sistemih različno, zato je možno, da se vizualna podoba malenkostno razlikuje (nisem imel časa za podroben test).


Na kratko o meni

Sem 20-letni študent gradbeništva. Zaradi novega šolskega sistema moram trenutno čakati nekaj mesecev, da bom lahko opravil izpit ki mi še manjka, zato sem se odločil, da si med tem poiščem delo ki me veseli. V prostem času se rad ukvarjam s programiranjem (Web, Visual Basic, Ruby), testiranjem varnosti Wifi mrež in testiranjem open source programov (sem član Ubuntu Linux U+1).
