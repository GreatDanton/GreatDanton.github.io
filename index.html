Dobrodošli!

Stran je bila na začetku namenjena samo zame, za učenje front end web-developmenta (testiranje css frameworkov, jquery modulov ipd). Ker trenutno iščem delo na področju web developmenta, sem se odločil da jo odprem za javnost (delodajalce), saj slika pove več kot 1000 besed. Stran sem napisal sam v html5/css3/jquery, uporabil sem le Twitterjev Bootstrap grid system. Spletna stran se prav tako odziva na različne resolucije (RWD). Ko je resolucija zaslona manjša kot 768px se glavni meni skrije in pojavi se nov način izbire podstrani.

Komentarji na podstrani Test so izključno samo prikaz javascripta, saj je potrebno za komentarje imeti bazo podatkov, katere še nisem implementiral. Prav tako je tudi sign-in link namenjen kasnejši implementaciji login sistema.
