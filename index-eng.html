Welcome

The page was in the first place created for me only, so I was able to learn the latest web developing tricks and try the latest web frameworks, jquery moduls etc. Since I am searching for work in web development field, I decided to bring this page to public, since a picture/website tells a thousands of words. Website was written by me in html5/css3/jquery, I used only Kube framework grid system. Website is able to adapt according to the screen resolution (responsive web design). When the screen resolution is less than 651px, the main menu become hidden, and you can choose pages with hamburger menu.

Comments on the Test page are for show-off only, since I would need database for storing them, which I haven't implemented yet. Sign in button was created for later as well. Since the same browsers on different operating systems render websites different, there is a possibility, that the same page might look slightly different, than it looks on my machine (I didn't have time for detailed test, nor the resources).
